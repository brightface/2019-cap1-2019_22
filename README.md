### 1. 프로젝트 소개
저희 M.Fi 팀은 WiFi CSI를 이용하여 사람의 동작 감지 서비스인 MDW를 만듭니다.<br/>
현재의 CCTV와 같은 광학적 감지장비는 시각적 제약이 있는 상황에서는 성능을 모두 발휘하기 어려울 뿐만 아니라<br/>
사생활 침해라는 부작용을 가지고 있습니다.<br/>
반면 CSI를 이용한 감지 시스템은 WiFi 채널의 변화를 감지하여 사람의 상태를 파악하는 기술이므로 시각적 제약이<br/>
없을 뿐 아니라 사생활 침해가 적다는 장점을 가지고 있습니다.<br/>
